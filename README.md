# Running the Project
- Clone repository
  > git clone https://github.com/inacio000/book-store.git
- Install dependencies
  > npm install

## Start Server
  >  json-server -p 3001 db.json
  
  `Or you may test the empty db running the command below` 

  >  json-server -p 3001 db-empty.json

## Run frontend
  > npm start
